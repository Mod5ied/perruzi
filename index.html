Why Peruzzi

Direct to the OS. Nothing in Between.

Peruzzi skips the middleware and talks straight to your operating system. No emulation layers. No latency. No abstraction tax.

Keyboard-Layout Blind

Core Engine

Peruzzi injects through CGEvent on macOS and SendInput on Windows at the Unicode level. Your active keyboard layout is completely irrelevant. QWERTY, AZERTY, Dvorak — it outputs the right character every single time.

One Key Kills It

Safety

Press ESC and injection stops. Full stop. The listener runs on a native event hook with zero idle CPU usage. No polling thread sitting in the background. No reaction delay. Instant.

You Set the Pace

Control

A precision slider lets you dial in a delay anywhere from 5ms to 200ms. Shift+Return handling keeps multi-line inputs from accidentally submitting mid-entry. You control every millisecond.

Looks Like a Human Typed It

Humanise Mode

Bigram-based speed variation, hesitation before punctuation, cognitive pauses at word boundaries, velocity curves, and occasional typo-then-correction loops. The kind of imperfection that reads as real.

3 Steps

Three Steps to Hands-Free Typing.

01

Load Your Text

Open Peruzzi from your menu bar on macOS or the main window on Windows. Paste in whatever you need injected: a script, a code block, a form response, anything.

02

Dial It In

Drag the speed slider to your preferred delay. Flip on Humanise Mode if you want natural variation in the output. Press Start and a 5-second countdown gives you time to get ready.

03

Hand It Over

Switch to the application you want to type into and place your cursor where you need it. Peruzzi takes it from there. Hit ESC at any point to cut it off immediately.

Quickest Right-Click to Override

Peruzzi is not distributed through the App Store. macOS will likely block the first launch. It takes about 20 seconds to fix.

  1. 1.Open Finder and find Peruzzi.app.
  2. 2.Hold Control and click the icon.
  3. 3.Choose Open from the menu that appears.
  4. 4.Click Open when the warning box comes up.
  5. 5.Peruzzi is now whitelisted. Every launch after this is a normal double-click.
Alternative Adjust via System Settings
  1. 1.Open System Settings from the Apple menu.
  2. 2.Select Privacy & Security from the sidebar.
  3. 3.Scroll down to the Security block.
  4. 4.Set "Allow apps downloaded from" to App Store and identified developers.
  5. 5.Click Open Anyway if Peruzzi was already blocked on a prior launch attempt.
Required One Permission Required
  1. 1.On first launch, click Open System Settings when prompted.
  2. 2.Go to Privacy & Security, then Accessibility.
  3. 3.Find Peruzzi in the list and toggle it on. One time only.

Peruzzi needs Accessibility access to inject keystrokes into other applications. This is a standard macOS requirement for any tool that interacts with other windows.

Windows No runtime required

Peruzzi on Windows needs nothing extra.

  1. 1.Download the Peruzzi.exe file.
  2. 2.Double-click it — no installer, no setup wizard.
  3. 3.No .NET, no Visual C++ redistribution, no setup wizard.
  4. 4.That is the entire Windows setup process.

Peruzzi is open source — you can inspect the full source code on GitHub before running it. If your antivirus flags the .exe, this is a false positive common to keyboard automation tools.
